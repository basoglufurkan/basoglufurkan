## Architecture Philosophy

```
Clean Architecture  →  separation of concerns is mandatory
Coordinator         →  lightweight, no business logic
Dependency Injection →  Factory pattern, no tight coupling
Testability         →  every layer should be mockable
```

> *"Clean architecture scales products. Good UX scales revenue."*
